# Python Syntax

Sintaks Python mendefinisikan serangkaian aturan yang digunakan untuk membuat Program Python. Sintaks Bahasa Pemrograman Python memiliki banyak kesamaan dengan Bahasa Pemrograman Perl, C, dan Java. Namun, terdapat beberapa perbedaan yang jelas antara bahasa-bahasa tersebut.

## Program Python Pertama
Mari kita jalankan program Python untuk mencetak "Halo, Dunia!" dalam dua mode Pemrograman Python yang berbeda. (a) Pemrograman Mode Interaktif (b) Pemrograman Mode Skrip.

## Python - Pemrograman Mode Interaktif
Kita dapat memanggil interpreter Python dari baris perintah dengan mengetik python pada command prompt sebagai berikut −

```bash
$ python3
Python 3.10.6 (main, Mar 10 2023, 10:55:28) [GCC 11.3.0] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>>
```

Di sini `>>>` menunjukkan Command Prompt Python tempat Anda dapat mengetik perintah. Mari ketik teks berikut pada prompt Python dan tekan Enter −

```bash
>>> print ("Hello, World!")
```
